## Installation üõ†Ô∏è

### Prerequisites
- Node.js >= 16.x
- PostgreSQL >= 12
- npm >= 8.x

### Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/BohBOhTN/storia-coffee.git
    cd storia-coffee
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Configure environment variables:
    ```bash
    cp .env.example .env
    ```
4. Run the application:
    ```bash
    npm start
    ```
